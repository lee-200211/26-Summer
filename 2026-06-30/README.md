# 오늘의 학습 내용

**날짜:** 2026-06-30

## 학습 요약

오늘은 Kotlin의 **클래스 상속과 함수 활용**을 중심으로 학습했다.

### 1. 클래스 상속과 다형성
- `Shape`, `Circle`, `Rect`, `Ellipse` 예제로 상속 구조를 이해했다.
- 부모 타입(`Shape`)으로 자식 객체들을 다루면서 **다형성**을 확인했다.
- `is`, `as`를 사용해 타입 확인과 다운캐스팅을 연습했다.

### 2. 추상 클래스
- `abstract class`는 객체를 직접 만들 수 없고, 상속을 위한 기반이 된다는 점을 학습했다.
- 추상 메소드는 반드시 하위 클래스에서 구현해야 한다.
- 일반 메소드와 추상 메소드를 함께 둘 수 있다.

### 3. 인터페이스
- `interface`를 통해 추상 메소드와 기본 구현 메소드를 함께 정의할 수 있음을 확인했다.
- `Duck` 예제를 통해 인터페이스 구현과 메소드 오버라이딩을 연습했다.

### 4. 클래스 위임
- `Printer by printer` 형태로 기존 구현을 다른 클래스에 위임하는 방법을 배웠다.
- 위임을 활용하면 중복 코드를 줄이고 역할을 분리할 수 있다.

### 5. 람다 함수와 고차 함수
- `{ it * 2 }` 같은 람다 표현식을 사용해 간단한 함수를 정의했다.
- 함수를 반환하는 `multiplier` 같은 **고차 함수**를 학습했다.
- 람다를 인자로 받는 함수와, 함수를 반환하는 함수의 구조를 익혔다.

## 느낀 점
- Kotlin은 클래스, 인터페이스, 람다를 조합해서 깔끔하게 코드를 구성할 수 있다.
- 특히 위임과 고차 함수는 코드 재사용성과 확장성을 높이는 데 유용하다.
