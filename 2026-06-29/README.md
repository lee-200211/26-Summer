# 오늘 학습한 내용

**날짜:** 2026년 6월 29일

이 디렉터리에는 `Array and for-Loops,classes.ipynb` 노트북이 있으며, 오늘은 Kotlin 기초 문법과 자료구조, 클래스 관련 내용을 학습했다.

## 학습 내용

- 배열 생성과 초기화
- `IntArray` 같은 기본형 배열 사용
- `for` 문과 `indices`, `withIndex()` 반복
- 람다를 이용한 배열 생성
- `List`, `Set`, `Map` 컬렉션
- 클래스 정의와 생성자
- 상속 관계와 `open` 키워드
- `data class`
- `companion object`

## 정리

배열과 반복문을 먼저 익히고, 컬렉션의 특징을 비교했다. 이후 클래스, 상속, 데이터 클래스, companion object까지 살펴보며 Kotlin의 기본 구조를 정리했다.
